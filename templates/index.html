How We Verify Claims

  • Google Fact Check API - Primary fact-checking database
  • Web Search Verification - Cross-references multiple sources
  • AI Analysis - Identifies verifiable elements in claims

Our Verdict System

  • Verified True - Confirmed by multiple reliable sources
  • Verified False - Proven incorrect by reliable sources
  • Partially Accurate - Mix of true and false elements
  • Unverifiable - Cannot verify with available information
  • Opinion - Subjective statement, not factual claim
