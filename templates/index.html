Services We Use for Fact-Checking

  • Google Fact Check API - Primary fact-checking database
  • Federal Reserve Economic Data (FRED) - Official economic statistics
  • OpenAI - AI-powered claim analysis
  • News APIs - NewsAPI & MediaStack for current events
  • Pattern Analysis - Identifies common false claim patterns
  • Wikipedia API - General knowledge verification
  • Semantic Scholar - Academic paper verification
  • CDC Data API - Health statistics and COVID data
  • US Census API - Demographic and population data
  • NOAA Climate Data - Weather and climate statistics
  • Context Resolution - Handles pronouns and temporal references
  • Speaker History Tracking - Maintains credibility records

How It Works

We extract factual claims from transcripts using NLP, then verify each claim against multiple sources. Each claim receives a verdict (true, false, misleading, etc.) with confidence scores and explanations.
